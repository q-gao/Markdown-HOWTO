## My Examples
Image vs. Clickable Image
- Image
```markdown
![CMake][CMake Logo]
```
- Clickable Image
```markdown
[![alt][CMake Logo]](http://cmake.org)
```
[CMake Logo]: https://cmake.org/wp-content/uploads/2014/06/cmake_logo-main.png

![CMake][CMake Logo]

[![alt][CMake Logo]](http://cmake.org)
